# Ferramentas para Instalação

## Spring Tool Suite

**Primeiro passo**
Abra o terminal e atualize os pacotes:
_sudo apt-get update_

**Segundo passo**
É atualizar os caminhos:
_sudo apt-get upgrade_

**Terceiro passo**
Se você usa a Snap store pode colocar o seguinte comando:
_sudo snap refresh_
